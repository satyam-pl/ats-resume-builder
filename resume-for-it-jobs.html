Resume Format for IT Jobs in India (TCS, Infosys, Wipro & Startups)

The IT sector employs over 5 million people in India. Whether you're applying to an IT giant like TCS or a fast-growing startup, your resume needs to speak the right language. This guide covers exactly what IT recruiters in India look for.

IT resume format — what's different

An IT resume is more technical than a generic resume. Recruiters and ATS systems scan specifically for technology stacks, tools, and project experience. Soft skills matter far less than hard technical skills in the first screening round.

Service companies (TCS, Infosys, Wipro, HCL)

These companies hire in bulk. Their ATS systems are highly automated. Here's what matters most:

  • CGPA above 6.5 (they often filter below this automatically)
  • No active backlogs
  • Clear skill keywords matching their job description
  • Clean, simple resume format — no fancy design
  • Mention any relevant certifications

For TCS NQT and Infosys InfyTQ, having certifications from their own platforms (TCS iON, Infosys Springboard) can give you an edge.

Product companies and startups

Product companies and startups look for different things:

  • Specific project experience with real impact
  • GitHub profile with actual code
  • Problem-solving ability shown through projects
  • Quantified achievements (numbers, percentages, scale)
  • Side projects, open source contributions, hackathons

How to write IT experience bullets

Use this formula: Action verb + Technology + What you did + Result

"Developed automated data quality checks using PySpark on Azure Databricks, reducing data errors by 35% and saving 8 hours of manual work per week."

Certifications that matter in IT India

  • Microsoft AZ-900 / AZ-104 / DP-900 (Azure)
  • AWS Cloud Practitioner / Solutions Architect
  • Google Professional Data Engineer
  • Databricks Certified Associate
  • Oracle Java Certification
  • Scrum / Agile certifications
