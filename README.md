## STEP 1: DEPLOY TO GITHUB PAGES (FREE)

1. Go to https://github.com and create a free account
2. Click "New repository"
3. Name it exactly: `cv-builder` (or any name you like)
4. Make it PUBLIC
5. Click "Create repository"
6. Upload all the files from this folder
7. Go to Settings → Pages → Source → "Deploy from branch" → main
8. Your site will be live at: https://YOUR-USERNAME.github.io/cv-builder/

---

## STEP 2: UPDATE YOUR URLS

Before uploading, replace "yoursite.github.io" in all HTML files with your actual GitHub Pages URL.

For example if your GitHub username is "satyam123":
- Replace: https://yoursite.github.io
- With: https://satyam123.github.io/cv-builder

---

## STEP 3: SUBMIT TO GOOGLE (VERY IMPORTANT)

1. Go to https://search.google.com/search-console
2. Add your site URL
3. Verify ownership (GitHub Pages — use HTML tag method)
4. Submit your sitemap: paste your sitemap URL
   Example: https://satyam123.github.io/cv-builder/sitemap.xml
5. This tells Google to index your site faster

---

## STEP 4: APPLY FOR GOOGLE ADSENSE

Wait until you have:
- At least 4-6 weeks of the site being live
- Some organic visitors (even 50-100/day is enough to apply)

Then:
1. Go to https://adsense.google.com
2. Sign in with your Google account
3. Add your site URL
4. Google will review your site (takes 2-4 weeks)
5. Once approved, replace the "Advertisement" placeholder divs in each HTML file with your actual AdSense code snippet

---

## STEP 5: WHERE TO ADD ADSENSE CODE

In each HTML file, find this line:
```html
<div class="ad-slot">Advertisement</div>
```

Replace the content inside with your AdSense ad unit code.
